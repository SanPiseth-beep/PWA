How to Use This Website

This website will randomly pick a set of color swatches to create a palette from an uploaded image.

Instructions:

  1. Click the "Upload Image" button on the homepage.
  2. Select an image from your computer.
  3. Wait for the image to load. The website will display the extracted color palette below the image.
  4. If you wish to upload another image, click the "Clear Image" button to reset the canvas.

Note:

The palette generated may not always perfectly match the image colors as the algorithm uses a simplified method to identify the dominant colors in a downsized version of the image (100x100 pixels) for faster processing.
